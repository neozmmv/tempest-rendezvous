# ⛈️ Tempest Server

Tempest is a secure and self-hosted messaging service.

The idea is to have this project running on Tor network for easy setup, no need to setup port-forwarding and router configs.

The project uses hybrid encryption (RSA + AES) to securely deliver messages and keep them safely stored so one but you can see.

## Still in development!

Run with `cargo run` or `cargo watch -x run`.
